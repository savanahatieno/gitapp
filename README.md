.

## Author
> Agutu Savanah.

## Installations
* nodejs
* npm
* angularCLI
* typescript
* bootstrap

## Setup
Run ``https://github.com/savanahatieno/gitapp.git``

or download the zip file from github.

After extracting the files, run 

* 1  .`` cd gitapp.`` 
* 2  .``ng serve.``

you can also run tests with ``ng test.``
